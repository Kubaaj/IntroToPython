Team members: Jakub Ajchel, Jerzy Bednarski

Our application will be an on-line video rental platform. Different users will be able to rent videos and pay for them on-line. 
Each user should have an account on the platform. According to his/her previous choices and other users' choices platform will 
recommend each user movies which suit him/her. 

Platform will consist of multiple parts:
- web application for customers with login window, search engine, recommendations and cart
- dashboard for platform's owner presenting statistics of video rental
- database with few thousands of movies with their descriptions

- payment system
